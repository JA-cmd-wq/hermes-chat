# 更新日志

## [v1.3.1] - 2026-04-21 18:40

### 🛠 修复 (Fix)
- **强制破除缓存**：给 `style.css` 和 `app.js` 添加版本查询参数 `?v=1.3.1`，配合 `Cache-Control: no-cache` meta 标签，确保用户手机/PC 浏览器每次都能拉到最新代码，不再被 CDN 或浏览器缓存锁死旧样式。

### 🎨 细节优化 (UX)
- **用户气泡瘦身**：
  - 改为 `width: fit-content` 精准贴合内容，不再留空白。
  - 最大宽度限制为 `min(560px, 80%)`，超宽内容自动换行。
  - 圆角从 18px 收敛到 16px（移动端 14px），视觉更干净。
- **移动端细化**：
  - 消息容器 padding 从 20px 收窄到 16px。
  - 字号自动降至 14.5px、行高 1.7，阅读更舒适。
  - 导航栏、输入框、页脚同步优化移动布局。
  - 用户气泡最大宽度限制到 85%，避免过宽。
