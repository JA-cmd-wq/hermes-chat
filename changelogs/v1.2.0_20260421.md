# 更新日志

## [v1.2.0] - 2026-04-21 18:25

### ✨ 视觉优化 (Visual)
- **去 Emoji 化**：移除所有 UI 中的 Emoji 符号（⚙️, 🗑️, 🔗, ❌, 👁, ⚠️ 等）。
- **极简 SVG 图标**：将功能按钮（设置、清空、显隐密钥、助手头像）全部替换为 Claude 风格的极简线型 SVG 图标。
- **排版纯化**：欢迎页文案及设置面板说明文字去除了符号干扰，视觉更加纯净。
- **错误提示优化**：移除错误消息气泡前的警告符号，改为通过颜色区分。

### 🛠 修复 (Fix)
- 统一了全局图标风格，确保在所有平台（Windows/Android/iOS）上的视觉表现一致，不再受系统 Emoji 渲染差异的影响。
