# 更新日志

## [v1.3.0] - 2026-04-21 18:32

### 🎨 布局重构 (Layout)

#### 消息列居中限宽
- 全局统一 `--chat-max-width: 740px`，消息区、输入区、底部信息三列完美对齐。
- PC 大屏下不再占满整个宽度，视觉更聚焦，符合 Claude 官方阅读体验。

#### 助手消息 Claude 风格化
- **移除助手消息的背景色块**：改为纯文本无背景（与 Claude 官网一致，像阅读文章）。
- **Hermes 作者标**：在消息顶部新增 "☤ Hermes" 小标签（极简线型 Caduceus 图标 + 作者名），取代之前浮动在气泡角落的小图标。

#### 用户气泡优化
- 气泡宽度从 75% 调整为 85% 上限，但由于整体限宽在 740px，实际更紧凑。
- 圆角从三角形调整为更柔和的 `18px 18px 4px 18px`。
- 内边距从 `12px 16px` 收窄至 `10px 16px`。
- **时间戳移出气泡外**：从气泡内白字改为气泡下方的淡灰小字（placeholder 色），不再干扰内容阅读。

#### 间距与层次
- 消息间 gap 从 16px 扩大至 28px，更透气。
- 底部信息字号减小至 11px + 淡化为 placeholder 灰，弱化非核心信息。

### 🛠 内部重构 (Refactor)
- `addMessage`、`addLoadingBubble`、`createAssistantBubble` 的 DOM 结构统一调整为：`.message-author` / `.message-bubble` / `.message-time` 三段式。
- `updateAssistantBubble` 改用 `.assistant-content` 选择器，避免用户/助手选择器冲突。
