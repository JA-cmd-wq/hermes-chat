# 更新日志

## [v1.0.0] - 2026-04-21

### ✨ 新增 (Feat)
- **项目初始化**：创建纯静态 Hermes Agent Chat UI。
- **UI 设计**：实现了 Claude 风格的暖赤陶色（Terracotta）调色板。
- **页面结构**：创建 `index.html`, `style.css`, `app.js`, `favicon.svg`。
- **核心功能**：
  - 支持 SSE 流式输出与打字机光标动画。
  - 支持 Markdown 渲染与代码高亮（Highlight.js）。
  - 提供响应式布局（手机端完美适配）。
  - 右侧滑入式设置抽屉（配置 API 地址、密钥及 System Prompt）。
  - 密码显隐、连接状态指示灯。
- **自动记录与部署**：建立基于 changelogs 的版本控制与自动 GitHub Pages 部署机制。
- **文档说明**：新增 `README.md`，提供使用与部署指南。
